# 宮町商店街 店舗検索サイト｜店舗データ更新ガイド

このサイトの店舗情報は、すべて1つのファイルで管理されています。
コードの知識がなくても、このガイドの手順で追加・修正・削除ができます。

---

## 店舗データの場所

```
src/data/miyamachi_stores.json
```

183店舗(2026年7月時点)の情報が、以下の形式で並んでいます。

```json
{
  "id": 12,
  "name": "宮町メガネ館",
  "category": "ショッピング",
  "area": "宮町",
  "address": "仙台市青葉区宮町5-2-10",
  "phone": "022-275-8013",
  "hours": "AM9:00-PM6:30",
  "closedDay": "日曜日",
  "services": "メガネ販売",
  "officialImage": "https://omiyamachi.com/...(画像URL)",
  "website": "https://...(任意・公式サイトやSNS)"
}
```

## 各項目のルール

| 項目 | 必須 | ルール |
|---|---|---|
| id | ○ | 他の店舗と重複しない番号。**新規追加時は「今ある最大のid + 1」** |
| name | ○ | 店舗名 |
| category | ○ | 次の7つのいずれか(表記を正確に): ショッピング / グルメ＆スイーツ / スーパー・食料品・お酒 / 暮らしのサービス / 医療・薬局・美容・健康 / 学ぶ・スクール / 遊ぶ・趣味・ビジネス |
| area | ○ | 次のいずれか: 宮町 / 東照宮 / 小田原 / 小松島 / 花京院 / 上杉 / 中江 / 梅田町 / 福沢町 / 高松 / その他 |
| address | ○ | 「仙台市青葉区」から書く |
| phone | ○ | ハイフン区切り。ない場合は「電話番号なし」 |
| hours | ○ | 例: AM10:00-PM7:00 |
| closedDay | ○ | 例: 水曜日・第2木曜日 |
| services | ○ | 取扱商品・サービスの説明(検索対象になるので具体的に) |
| officialImage | 任意 | omiyamachi.com掲載の店舗写真URL |
| website | 任意 | 公式サイト・SNS of URL(httpsから) |

※ categoryとareaは**一字一句正確に**。表記が違うと絞り込みに表示されません。

## 更新のやり方(推奨: AI Studioで指示するだけ)

一番簡単な方法は、Google AI Studioでプロジェクトを開き、Geminiのチャット欄に日本語で指示することです。

**例1: 営業時間の変更**
> 宮町メガネ館の営業時間をAM9:30-PM6:00に変更してください

**例2: 新しい店舗の追加**
> 以下の店舗をmiyamachi_stores.jsonに追加してください。idは最大値+1で。
> 店名: ○○食堂 / 業種: グルメ＆スイーツ / エリア: 宮町 /
> 住所: 仙台市青葉区宮町X-X-X / 電話: 022-XXX-XXXX /
> 営業時間: AM11:00-PM8:00 / 定休日: 月曜日 / 内容: 定食・丼もの

**例3: 閉店した店舗の削除**
> ○○(店名)をmiyamachi_stores.jsonから削除してください

## 公開までの流れ

1. AI StudioでGeminiに変更を指示(またはCodeタブで直接編集)
2. Previewタブで表示を確認
3. **Settings(歯車) → Export to GitHub** でエクスポート
4. 約2分でサイトに自動反映(GitHub Actionsが自動でビルド・公開)

反映状況は https://github.com/abebanjp-glitch/miyamachi-shop-search/actions で確認できます。
緑のチェックが付けば完了です。

## 困ったとき

- **反映されない** → スマホ・PCで再読み込み(キャッシュが残っている場合があります)
- **絞り込みに出てこない** → categoryかareaの表記ゆれを確認
- **Actionsが赤い×** → 同ページの「Re-run all jobs」で再実行(一時的なエラーのことが多い)
